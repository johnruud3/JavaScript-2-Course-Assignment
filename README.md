# Feed Me

A social media platform where users can create, read, and interact with posts. Built as the JavaScript 2 Course Assignment at Noroff.

## Features

- Browse and search posts from the community
- Create, edit, and delete your own posts
- React (like) and follow other users
- User registration and login with JWT authentication
- Responsive design using Bootstrap 5

## Prerequisites

- A modern browser with ES module support
- A local web server (required for ES6 modules — e.g. [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code)
- A `@stud.noroff.no` email address to register an account

## Getting Started

### Running the project

1. Clone or download this repository:

```bash
git clone https://github.com/John-Kristian/JavaScript-2-Course-Assignment.git
```

2. Open the project in VS Code and start Live Server, or use any local HTTP server:

```bash
npx serve .
```

3. Open `index.html` in your browser through the local server.

## Environment Variables

No `.env` file is needed. The API key is included in the source for academic purposes.
